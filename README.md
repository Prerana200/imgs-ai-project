Dakshiya AI
Intelligent Industrial Monitoring System
---------------------------------------
Real-time AI monitoring for industrial environments

Dakshiya AI is an intelligent industrial monitoring system designed to
analyze real-time video from manufacturing environments using AI.

The system detects foreign objects, monitors material size, and provides
real-time alerts to improve industrial safety and process efficiency.

## Features
- Real-time object detection
- Process monitoring
- Analytics dashboard
- Alert system
- Data visualization

## Tech Stack

- Python
- FastAPI
- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV
- Docker
- SQLite
